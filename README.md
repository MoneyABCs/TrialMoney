
# moneyabcs.org
Website for financial literacy

This website is developed using mean stack
#MEAN stack : (MongoDB, Express, Angular.js, Node.js)
1. This website is to spread financial literacy among Americans.
2. The article section in the home page has an algorithm that makes sure articles are collected 
from several websites and are fresh, and about key words that are most popularly searched on the web.
3. There is a login feature.
4. Logged in users can add articles to their dashboard.
5. You can email yourself articles.
